# Spike outputs

AHM spikes are exploratory feasibility studies. Where a spike produces
content that's commercially sensitive — wallet-level characterisations,
proprietary methodology, competitor analysis, taxonomy work, or similar —
that output lives in the private ahm-internal repo rather than this public
repo.

Spikes producing content that doesn't meet the sensitivity criteria can
continue to be committed here.

The decision is content-based: would this give a competitor or hostile
actor material insight into AHM's methodology, intelligence, or unreleased
thinking? If yes, it goes private. If no, it can stay public.
